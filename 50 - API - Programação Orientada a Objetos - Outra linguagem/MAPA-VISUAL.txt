🔗 ENDPOINTS DA API (12 no total)
═══════════════════════════════════════════════════════════════════════════════

GET REQUESTS:
  ✅ GET  /api/produtos                    → Listar todos
  ✅ GET  /api/produtos/:id                → Obter um
  ✅ GET  /api/produtos/:id/descricao      → Descrição
  ✅ GET  /api/produtos/tipo/:tipo         → Filtrar por tipo
  ✅ GET  /api/produtos/estoque/baixo      → Produtos com baixo estoque
  ✅ GET  /api/produtos/info/eletronico/:id   → Info específica (com cálculos)
  ✅ GET  /api/produtos/info/roupa/:id        → Info específica (com recomendações)

WRITE REQUESTS:
  ✅ POST   /api/produtos                  → Criar novo
  ✅ PUT    /api/produtos/:id              → Atualizar
  ✅ DELETE /api/produtos/:id              → Deletar
